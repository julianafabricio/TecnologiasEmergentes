# TecnologiasEmergentes
Projeto desenvolvido nas aulas de Tecnologias Emergentes 
🛍️ EcommerceBemBarato
EcommerceBemBarato é uma aplicação web desenvolvida com o framework Flask que simula uma plataforma de anúncios e vendas de produtos variados. Usuários podem se cadastrar, fazer login, criar anúncios de produtos, editar, remover, e visualizar categorias específicas como Eletrônicos, Casa e Decoração, Cama e Banho, entre outros.

🎯 Objetivo da Aplicação
O objetivo principal do EcommerceBemBarato é oferecer uma plataforma simples e funcional onde usuários possam:

- Criar e gerenciar seus próprios anúncios de produtos;
- Visualizar produtos por categoria;
- Realizar cadastros, logins e logouts com segurança;
- Acompanhar perguntas, compras e vendas;
- Simular um ambiente de e-commerce, ideal para estudos, testes ou MVPs de ideias futuras.

⚙️ Tecnologias Utilizadas
A aplicação foi construída com as seguintes tecnologias:

- Python 3
- Flask – Framework web leve e rápido
- Flask-Login – Gerenciamento de autenticação de usuários
- Flask-SQLAlchemy – ORM para interação com banco de dados
- MySQL – Banco de dados relacional
- Jinja2 – Motor de templates integrado ao Flask
- HTML5 / CSS3 – Para a construção das páginas
- Hashlib (SHA-512) – Para criptografia de senhas dos usuários
